 # 失落的亚特兰大 - 设计文档

 ## 项目定位
 - **游戏类型**：丧尸末世题材文字生存游戏，事件驱动叙事
 - **开发工具**：Godot 4.x（GDScript）
 - **目标平台**：手机 + PC（竖屏优先 720×1280）
 - **性质**：个人项目 / Demo，不涉及商业变现和版号问题

 ## 核心玩法
 - **三项资源**：食物 / 精神 / 体力
 - **每日流程**：白天自由行动（3个行动格：搜索/休息/探索/加固）→ 夜晚强制事件 → 日结消耗资源 → 判断胜负
 - **胜利条件**：生存 30 天
 - **事件系统**：JSON 配置驱动，事件之间通过浅层分支链（next_event_id / unlock_event_id）关联
 - **道德选择**：不做隐藏数值，通过后续事件文本引用来反映玩家历史选择

 ## 美术风格
 - 轻量图文混合，AI 生成场景氛围图
 - 深色调 UI，文字为主，图作为氛围点缀

 ## 存档
 - 自动存档单档位，每晚事件结束后保存，死亡后删档
