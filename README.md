# 失落的亚特兰大 Demo

《失落的亚特兰大》是一款丧尸末世题材的文字生存 Demo，使用 Godot 4.x 和 GDScript 开发。

项目当前目标是先完成一个可扩展的核心框架，再逐步加入事件、天气、背包、商店等系统。

## 项目定位

- 类型：文字生存 / 事件驱动叙事
- 引擎：Godot 4.x
- 平台：PC + 手机
- 画面：竖屏优先，目标分辨率 720x1280
- 性质：个人 Demo

## 核心玩法

玩家需要在末世环境中管理资源并尽可能生存更久。

当前设计中的核心资源：

- 食物
- 精神
- 体力

每日流程规划：

1. 白天选择行动
2. 结算行动结果
3. 触发夜晚事件
4. 执行日结消耗
5. 判断是否继续生存

目标胜利条件暂定为生存 30 天。

## 运行方式

1. 使用 Godot 4.x 打开项目目录。
2. 打开 `project.godot`。
3. 运行主场景。

主场景已配置为：

```text
res://scenes/MainMenu.tscn
```

## 开发约束

本项目虽然是 Demo，但后续代码必须保持可扩展性。

核心约束见：

- `开发规范.md`
- `AGENTS.md`

后续新增功能应遵守以下方向：

- 围绕全局阶段推进流程
- 通过 Hook 或事件解耦系统
- UI 只负责展示与输入
- 存档按系统拆分
- 天气、背包、商店等功能作为独立系统接入
